## 데이터 프레임(표만들기)
- 각각의 함수는 같은 수의 집합을 가지고있어야한다.
- 함수 = data.frame(a, b, c) a, b, c에 해당하는 열을 가진  frame을 만든다

```r
no= c(1, 2, 3, 4, 5)
sex= c("male", "female", "male", "male", "female")
korean= c(87, 92 , 95, 81, 87)

exam = data.frame(no, sex, korean)
exam
```

```
##   no    sex korean
## 1  1   male     87
## 2  2 female     92
## 3  3   male     95
## 4  4   male     81
## 5  5 female     87
```
- 함수$내용 = 함수속에 있는 내용을 vector로 접근

```r
exam$korean
```

```
## [1] 87 92 95 81 87
```

- 함수[행,열] = 행, 열에 속해있는 내용을 출력

```r
exam[2,3]
```

```
## [1] 92
```

- 함수[a:b, c:d] = a행~b행, c열~d열에 속한 내용을 출력

```r
exam[1:3, 1:3]
```

```
##   no    sex korean
## 1  1   male     87
## 2  2 female     92
## 3  3   male     95
```

- 함수$내용[n] = 함수속에 있는 내용 중 n번째의 것을 vector로 접근

```r
exam$korean[2]
```

```
## [1] 92
```

- str(함수) = 함수 속 모든 vector의 datatape과 class를 한번에 접근 

```r
str(exam)
```

```
## 'data.frame':	5 obs. of  3 variables:
##  $ no    : num  1 2 3 4 5
##  $ sex   : chr  "male" "female" "male" "male" ...
##  $ korean: num  87 92 95 81 87
```

### 데이터 추가
- 새로운 함수를 정의해 준 후 원래의 함수 + 추가된 함수로 만든다

```r
no= c(1, 2, 3, 4, 5)
sex= c("male", "female", "male", "male", "female")
korean= c(87, 92 , 95, 81, 87)

exam = data.frame(no, sex, korean)
exam
```

```
##   no    sex korean
## 1  1   male     87
## 2  2 female     92
## 3  3   male     95
## 4  4   male     81
## 5  5 female     87
```

```r
english = c(95, 77, 65, 92, 99)
exam_a = data.frame(exam, english)
exam_a
```

```
##   no    sex korean english
## 1  1   male     87      95
## 2  2 female     92      77
## 3  3   male     95      65
## 4  4   male     81      92
## 5  5 female     87      99
```
### 데이터 제거
- 함수[-n, -m] = n행 m열의 내용을 제거
- 함수[-c(n,m), -c(a,b)] = n행과 m행, a열과 b열을 제거

```r
exam_a[-2,-3]
```

```
##   no    sex english
## 1  1   male      95
## 3  3   male      65
## 4  4   male      92
## 5  5 female      99
```

```r
exam_a[-c(1,3), -c(1,2)]
```

```
##   korean english
## 2     92      77
## 4     81      92
## 5     87      99
```
